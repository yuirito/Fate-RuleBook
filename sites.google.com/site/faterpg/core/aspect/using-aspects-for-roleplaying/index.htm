使用形象做角色扮演

最後，形象有著你可以從每個遊玩實例中借鑒的被動使用方式。玩家們，你可以用它們作為扮演你的角色的導引。這應是不言自明的，但是我們總要說明一下，在你的角色表上的形象在任何時候都是你的角色的真實，而不只在它們被援引或強迫的時候。
將你的形象的集合視作像塔羅牌或茶葉一樣的隱諭。它們給予你的角色的樣子一個大方向，並且你可以從字句發現它們揭露出有趣的含義。如果你想知道你的角色在特定情況下可能會做什麼，看看你的形象。它們是怎麼描述你的角色的性格、目標和願望？你的形象有沒有暗示行動方向的任何線索？一旦你找到，就去做吧。
用你的形象做扮演也有另一個好處：你餵給遊戲主持人使用強迫的點子。你已經把你的形象帶到遊戲中，這樣他所要做的就是為你提供困境而你也已經準備好接受了。
遊戲主持人，你會以相同方式使用你的非玩家角色的形象，但你有額外的「解讀茶葉」的方式，你也可以用它們作為釐清世界是如何回應角色們的一種方式。是否有人擁有世界上最強壯的人的形象？一旦人們知道這點且做出回應時，這角色的聲譽可能會比較好。當角色路過時，人們可能會包圍爭睹。
此外，它也表明了一些關於角色的物理尺寸和體格。你知道，大多數人在擁擠處將會對該角色敬而遠之，可能是自然地威懾或可能因為被嚇到而顯得過於激動或粗暴。但沒有人會忽略該角色。插入這類與形象相關的細節到你的敘述中可以讓你的遊戲看起來更加生動和一致，即使沒有使用到命運點數。
在鋼鐵之心其中一場聚會中，蘭登回到了他的家鄉酒田鎮，發現它被野蠻人侵襲，他的導師老芬恩被綁架。
阿曼達告訴他，其他村民對於他回到家鄉都感到欣喜，而在一個與村中長老談話的場景中，她也說村民們要求他留下來幫助重建這個城鎮。
藍尼看看蘭登的一些形象：象牙帷幕門徒，我欠老芬恩一切，粗野禮儀，砸爛永遠是個選擇。他認為這些形象顯示蘭登是非常直接（粗魯無禮的）、積極、傾向用暴力解決問題並對他所屬非常忠誠。
由於他的形象，只要老芬恩仍然還有一絲希望，蘭登絕對不會留下來幫助這個城鎮。不僅如此，他還會告訴長老他對於老芬恩被綁走後他們卻沒有派出救援的感覺。他可能會用「懦弱」和「無能」這樣的詞語。你懂得，那種能真正讓人們同情你的話語。
阿曼達說他激怒了長老，因為他的無禮，他們正在考慮將他逐出城鎮。她拿著一個命運點數笑了起來，表示強迫，他的禮貌將讓他被踢出酒田鎮。
藍尼收下點數，接受那個困境。，「去他們的」他說著「我會救回老芬恩的，不用他們幫忙。」

移除或更改形象 
遊戲和角色形象會隨著發展變化。請看長期遊戲中的里程碑一節。 
如果你想擺脫某個局勢形象，你可以做下面兩件事之一：為了擺脫該形象特別擲骰作克服行動，或是擲骰作一些其他成功後能讓該形象失去合理性的行動。（例如，你正被擒抱，你可以嘗試逃開。如果你成功了，對你來說被擒抱就不合理了，所以你擺脫掉這個形象。） 
如果一個角色可以干涉你的行動，他們如同正常般得到擲骰對你主動對抗的機會。否則，遊戲主持人，設定被動的對抗就是你的任務，或者是沒有甚麼風險或有趣的結果時讓玩家無須擲骰就擺脫形象。 
最後，如果任何時候局勢形象在遊戲中沒有任何意義時，就擺脫它。

在遊戲中創造與發掘新的形象
除了你的角色形象、遊戲形象與遊戲主持人呈現的局勢形象，你在遊戲中可以創造、發掘或是取用其他形象。

給主持人

超強必殺主持技巧，如果你沒有為場景或非玩家角色製作任何形象，只要在它們擲骰創造優勢時問玩家想找的是什麼類型的形象。如果他們平手或成功，只需寫下類似於他們正在尋找的東西並說他們是對的。如果他們失敗，還是將它寫下來或者寫下另一個對他們來說不利的形象，以便與他們的期望相比。
在大多數情況下，你將用創造優勢行動來製造新的形象。當你描述一個給予優勢的動作，如果它需要一個新形象或是要從既有的形象衍生，那麼文字內容應該會告訴你。如果你把一個新狀況帶入遊戲中，像是對他人眼睛拋砂，這表明你需要一個新的局勢形象。
某些技能可以合理地使用已經存在於某人角色表上的形象來獲得優勢。在這種情況下，作為目標的玩家角色或非玩家角色將會主動對抗來阻止你使用該形象。
如果你不是想要免費援引，而只是覺得如果遊戲中有個特殊的局勢形象非常合理，你不需要擲骰子或做任何事情來製造新的形象，只需要提出建議，如果團隊認為他們是有趣的，就把它們寫下來。

秘密或隱藏的形象
有些技能還能讓你使用創造優勢行動來揭露隱藏的形象，無論是非玩家角色身上的或是在環境中的，在這種情況下，如果你擲骰的結果是平手或更好，遊戲主持人乾脆地告訴你那個形象是什麼。如果你不是很清楚知道該找什麼，可以利用這個方式「釣出」形象，夠好的擲骰結果就足夠當作找出某些優勢的理據支持。
一般來說，我們假設遊戲中大部分形象對玩家是公開的知識。玩家角色的角色表是放置在桌上的，而重要與配角的非玩家角色可能也是。這並非意味著角色總是了解這些形象，但這是創造優勢的行動存在的原因之一，幫助你證明角色如何了解其他角色。
此外，請記住形象只有在你使用它們時才能幫助加深故事，那些從來沒有被發現的形象就和一開始就不存在一樣。所以大多數時候，玩家應該總是知道哪些形象可供其使用，而如果有不確定角色是否知道的問題，使用骰子來幫你決定。
最後，遊戲主持人，我們知道有時候你會想保留非玩家角色的形象當作秘密，又或是現在還不想透露某些局勢形象，因為你試圖在故事中營造氣氛。如果玩家角色正在調查一個連續謀殺案，你不會希望有著反社會連環殺人犯形象的罪魁禍首的索引卡在冒險開始前就讓玩家們看到。
在這種情況下，我們建議你製作形象的內容時不要太直接與想保密的內容相關，但是要讓形象的細節內容在秘密被揭開時顯得合理。
阿曼達正在製作一個暗地裡是個吸血鬼的非玩家角色，這是她計劃中的劇段裡的主要壞人。而他也是玩家角色所要前往的城鎮的治安官，所以她不想太輕易地泄露真相。所以她不是做暗地裡是吸血鬼的形象，而是決定做一些個人細節：常年的夜貓子，比外表看來更強，事情錯綜複雜。如果玩家角色發掘出其中幾個或者在桌上看到它們，他們可能會開始懷疑這個非玩家角色，但又不會馬上毀掉這個劇段的謎團。
